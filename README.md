# Reno SEO Automation & Technical Audit Tool 🏔️💎

Hi, I'm **Cap Puckhaber**, founder of **Black Diamond Marketing Solutions** based in Reno, Nevada.

I built this tool to provide our clients with total transparency. Most SEO audit tools are "black boxes"—this Python script automates technical audits (Core Web Vitals, metadata gaps, and link health) so we can focus on high-level growth strategy for our Nevada partners.

### 🚀 Key Features
- **Automated Metadata Scraper:** Instantly flags missing H1s, Meta Descriptions, and Alt-tags.
- **Reno Local SERP Checker:** Simulates searches from local Reno/Tahoe IP addresses to track Map Pack performance.
- **Competitor Gap Analysis:** Compares site structure against the top-ranking local competitors.

### 🛠️ Quick Start
1. Clone this repo.
2. Run `pip install -r requirements.txt`.
3. Execute `python audit_engine.py --url https://yourwebsite.com`.
